## Prerequisites
- A Vocera account
- Retell API credentials

## Setting Up Retell Integration

### 1. Accessing Organization Settings
1. Log in to your [Vocera Dashboard](https://dashboard.vocera.ai)
2. Click on the **Organization** button in the navigation menu
3. Navigate to the **Settings** tab

<img src="/images/org-button-click.png" alt="Organization Settings Navigation" />

### 2. Configuring Retell Authentication
1. Locate the **Retell Integration** section
2. Enter your Retell API key in the designated field
3. Click **Save**

<img src="/images/retell/1.png" alt="Retell API Configuration" />

## Testing Your Agent

### 1. Select Retell as your provider when creating a new agent

 1. Navigate to the **Agents** section
 2. Click **Add Agent** button
 3. Select **Retell** as your provider
 4. Copy your agent id from Retell into the Assistant ID field

<img src="/images/create-agent.png" alt="Retell API Configuration" />
